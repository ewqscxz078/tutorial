		三之一 : 是不想做還是不會做?
			一個新的目標對員工來說
				1.代表更高的目標達成
				2.走出舒適圈面臨更大的挑戰 : 跨出舒適圈
				3.需要有新的工作方法
				4.更難達成的業績目標: 績效目標難度提高
				5.更多學習、更長工時 : 工作工時延長
			一般主管問法:有沒有什麼問題
				只會聽到: 有/沒有
			教練式的領導說法
				開放式的問題
					1.了解員工對目標認同程度
					2.遇到什麼疑慮或困難
					3.敢不敢提出
				要完成這個目標，你有幾成把握
					員工必須回答幾成
						=> 員工 : 大概五六成
							教練式的領導說法
								差那四五成是什麼原因?
								=> 了解員工信心與認知程度
								員工 : 因為工作太多
									手邊有哪些工作、專案
									這些事情佔了你多少時間?
								員工 : 目標太挑戰
									我想知道你以前怎麼完成目標的?
								員工 : 低頭沉默不語
									看起來你對目標似乎不是很有信心
									可以告訴我原因嗎?

									教練: 你希望我怎麼幫你?

				教練式的管理者需要分辨
					員工不能
						知識 / 經驗 / 專業
					員工不願意
						性格 / 價值觀 / 對管理團隊的看法
				與部屬的教練對話中
				要學會享受對談中短暫的沉默

		三之二 : 是不能給答案還是引導找到答案?
			核心技巧 : 提問
				1.了解問題來龍去脈
				2.掌握更多訊息
				3.引導啟發員工思考、跳出框架
				4.了解員工的疑慮或困難
				5.引導部屬逐步找到解決方案

				錯誤範例
					我是教練型主管，不會給你答案的
				正確範例
					1.這個問題我現在沒有答案，但我們討論一下，我相信可以找到答案
					2.這問題我有一些答案，但我想先聽聽你怎麼看這個問題

			GROW MODEL
				Goal				Reality				Options				Way Forward /Will
				目標				現況				可能的				行動
				談完後				現狀、問題			解決方案是什麼?		共同的結論以及
				希望得到什麼?		及原因是什麼?							行動計劃是什麼?

			範例
				目標1 : 談完希望得到什麼?
					1.我們現在要討論的問題是什麼?想得到的結論是什麼?
					2.待會我們談完後，你希望可以得到什麼?
					3.所以你今天來找我談的目的是什麼?
					4.我想花20分鐘時間，跟你談談如何達成下個月業績的方法，以及你需要我協助的地方
				目標2 : 談完希望得到什麼?
					1.你的目標夠挑戰嗎? 合理嗎? 為什麼?
					2.你希望何時完成這目標?
					3.如果不完成這目標會怎樣?
					4.怎麼知道你已經完成目標?

				現況1: 現狀、問題及原因是什麼?
					1.請談談目前狀況?
					2.針對這問題(或這件事)，你做過哪些努力?試過哪些選擇?結果如何?
					3.針對現在的情況，你的看法(或感受)是如何?
					4.對於現況你看那些機會?又有哪些可能的挑戰?
					5.你以往是如何讓自己工作更有效率的?
					6.請說說你(或別人)過去相關的成功經驗?你的學習是什麼?你會如何應用到未來的工作上?
				現況2: 現狀、問題及原因是什麼?
					績效表現有落差
						1.你的進度(成果)和你的目標差了??，原因是什麼?
						2.景氣問題的確認大家壓力很大，但一樣的景氣因素，
							有人可以達成目標，你覺得原因是什麼?
						3.哪個人(或哪個部門、哪件事)妨礙你把工作做好?
						4.除了上述因素，有什麼是你可以做的更好而沒有做的?
						5.還有其他因素嗎?

				選擇: 可能的解決方案是什麼?
					1.根據你剛剛的分析，你覺得有哪些方法可以幫你達成目標?
					2.還有其他可行的方法嗎?
					3.要解決這個問題，一般有下列三種方法，你覺得哪種較好?為什麼?
					4.這些方法的優缺點是什麼?
					5.你會如何排序這些方法的有效性(或可行性)
					6.如果你剛剛提到的產品(或流程)問題解決了，你接下來該做什麼?
					7.對於完成這件工作，你有幾成把握?差的那幾成是因為什麼?
					8.你希望我怎麼幫你?
					9.你還需要誰的幫忙?

		三之三 : 是問結果還是過程?
			GROW MODEL 四個步驟時間分配
				Reality 步驟占整個 GROW過程一半以上
					否則會錯失探討原因
					接受部屬片面的理由

			教練對話過程自然方法
				1.多練習，熟能生巧
					Practice makes perfect
				2.抓住部屬說的話之中的關鍵字
				3.善用對話過程短暫的沉默
					主管:除了大環境因素不佳之外有什麼是你該做沒做好的
					員工:沉默
					主管:沒關係，你想一想，我等你
				4.運用同理心與肯定的技巧
				5.給選擇題引導部屬思考
					若員工沒有想法
						1.給一半答案
						2.這個問題解決方法通常有三種，你覺得那些比較好?

		四之一 : 問題八: 是同理還是同意?
			同理心與肯定、支持
				過程
					1.討論有溫度
					2.有效降低對方的防備心
				技巧
					1.說出對方經歷的事實
					2.說出對方的感受
				同理
					同理不等於同意
					部屬真正要的不是主管同意他
					而是主管懂他
				肯定
					1.明確誠懇
					2.謝謝對方的表達也是一種肯定
		四之二 : 問題九: 留住員工要靠金錢、成就感還是歸屬感?
			員工屬性與情境領導
				動機/態度高、能力高
					良駒
						能力/經驗都強
						工作態度也積極正向
						很有企圖心
						=>	獎酬
							提供發展機會
							舞台
							授權
						通常不太需要詢問細節
						比較多話題
							是未來職涯規劃是什麼?
							有沒有符合你的規劃的進度?
								有?是什麼?
								沒有?是什麼?
							未來若要協助你達到未來職涯規劃身為主管的我可以幫你什麼或做什麼?
							你未來可以做什麼?
							你的表現已經很好了，可不可分享一下?是什麼能力或是你做對了什麼讓你表現這麼好?
							如果你現在手邊有個任務，本來是我做的，你覺得可以讓你承擔這個責任的話，你覺得這會是什麼工作?

							如果未被滿足
								1.被競爭對手挖角
								2.良駒變璞玉(態度變消極)

				動機/態度低、能力高
					璞玉
						能力/經驗都強
						甚至可能比主管都要強
						但心態保守
						有點倚老賣老
						=>
							1.直接回饋、表達感受
							2.交心/尋求對方協助
							3.適度壓力
							4.授權
							5.獎酬
						性質:被同理、被肯定、交心
						如何能將一身武藝得到更大的發揮?
						在這個部門妳比我資深，經驗也比我多，今天如果你是我這個位置你會怎麼看這件事情?

				動機/態度高、能力低
					蠻牛
						能力/經驗不足
						但態度正向
						這屬於欠裁培
						=>
							1.目標設定
							2.指導及示範
							3.經常性績效回饋
							4.教育訓練
							=> 極高機率會改善
				動機/態度低、能力低
					朽木
						能力/經驗都弱
						工作態度又差
						消極擺爛
						=>
							1.直接告知績效問題
							2.目標設定
							3.教育訓練
							4.經常性績效回饋
							5.凍結晉升、薪資
							6.輔導轉職、轉業
							7.解雇
							=> 較低機率會改善

		四之三 : 問題十: 是創造追隨者還是領導者?

			3P舊世代的價值觀與工作觀
				Pay Check 薪資
				Promotion 晉升
				Pension	  退休金
			3F新世代的價值觀與工作觀
				Friendly 友善
				Flexibility 彈性
				Future 未來

			創造更多領導者
				1.方法技巧是否有用對
				2.尊重部屬工作價值觀(3P、3F)
				3.平衡組織需求與員工期待

			教練式領導需思考的問題
				1.是員工難帶，還是主管不會帶? 如何分辨是前者還是後者?
				2.是要直接指導還是教練?哪種員工指導要多一些?哪種員工教練要多一些?
				3.是問題員工還是員工問題?前者員工已被認定有問題,這樣的認定會有什麼風險?
				4.是員工聽不懂還是主管說不清?SMART法則可以怎麼改善這一部分?
				5.員工是不想做還是不會做?前者該如何處理?後者又該如何回應?
				6.是不能給答案，還是引導對方自己找到答案?若對方一直沒有答案，怎麼辦?
				7.在教練員工時是要問結果還是過程?還是兩者皆是?為什麼?
				8.同理等於同意嗎?若不同，差別在哪裡?
				9.留住某個員工要靠金錢?成就感?還是歸屬感?如何透過教練技巧找出答案?
				10.一個真正的領導者，是要創造更多的追隨者還是創造更多的領導者?
		五之一:實例:被退回的請款單
			情境:協理下有經理、經理下有助理
				Q:有次活動後經理將收集後的發票收齊後交給協理轉給財會部但被退單
					因為60張裡有兩張沒有統編
					協理問經理為何有兩張沒有統編，經理答他授權給助理，且協理您有簽名，怎沒有確認呢?

					教練式提問
						1.確認雙方交接上認知一致
						2.確認對目標成果認知是否一致
						3.(協理/經理/助理)是否有做最後檢核?如何做?
			重點提醒:
				1.仔細請聽部屬的回答，從中找到關鍵字往下問
					例如:你對授權定義是什麼?
				2.透過提問，引導部屬察覺自己的盲點
					例如:是否只授權而沒有教導
				3.不須因為部屬中明顯或隱含的挑釁而動氣
		五之二:實例:每天走6000步不好嗎?

				主管面臨上有公司政策配合、員工不配合

					3.有沒有什麼方法提高員工對這個的活動認同
					4.該活動與員工 KPI 有關係? 主管答無關

					1.是否是員工職能所在?
						如何提高員工認同度?
					2.主管本身的認同程度?

			重點提醒
				1.如果主管自己都不認同的事，請勿期待底下的部屬會認同
				2.KPI目標達成和職能/行為的展現不要混為一談
				3.與其硬碰硬，不如先了解部屬的抗拒之後，再找出雙方都能接受的方法

		五之三:實例:工程師到底想要什麼?
			情境:公司某部門有個基層工程師，該工程師有些特殊的能力，有個項目在大陸需要其能力幫忙專案
				需要其工程師到大陸常駐
				該工程師表示因為快結婚了，未婚妻不想去大陸，因此回公司說沒法去
				該主管該如何處理該事情?
					主管嘗試 : 攜帶眷屬、加薪、升職，強迫去他就說會離職
					後來主管發現該工程師的罩門，喜歡被尊重的感覺
					因此主管透過層層上報，於工程師結婚當天，該集團總裁到場祝賀
						後來基層員工跟主管說:我會說服老婆一起去大陸
						因為這樣主管解決了這個問題
			重點提醒:
				1.金錢、成就感、歸屬感是留住員工的三大因子，三者有其一，就有機會留住員工
				2.透過觀察與提問，了解員工真正想要的是什麼
				3.很多時候，員工要的不是主管同意他，而是主管懂他

		五之四:實例:職缺開兩年都找不到新人
			情境:有個公司招人不是找不到對的人，而是找不到人
				他是品管部門的主管，他做的醫療器材很特殊，需要高要求品質
					想請品管工程師，但仍找不到
				問題:因人力缺口無法補足，主管必須親力親為，開出職缺已兩年
					細問其職缺要求的能力，過多的要求，即使自己訓練一個新人也要兩年
					因此是否降低標準，尋找願意學習的?

		五之五:實例:Top sales or Top manager
			情境:有個銷售學員去知名銷售天王學習銷售技巧:關心顧客、追蹤顧客
				正直誠信，甚至每年定期逢年過節或重要假日寫卡片86000張給顧客感謝或關心問候
				他坦言最後一點做不到
			重點提醒:
				1.Top sales 的成功非偶然，但 Top Manager
					可以讓部屬的成功成為必然
				2.教練是領導的目的，不是讓主管複製另一個自己
					而是啟發部屬看到自己的優點及限制，進而成為更好的他自己
				3.真正的領導者是要創造更多的領導者而非追隨者
