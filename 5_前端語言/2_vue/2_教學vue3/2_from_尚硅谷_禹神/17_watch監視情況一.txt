作用: 監視資料的變化
特點: 監視四種情況
	1. ref 定義的資料
	2. reactive 定義的資料
	3. 函數返回一個值(getter 函數)
	4. 一個包含上述內容的數組

比如使用情境
	1. age 大於 18歲，做對應的事情
	2. order 訂單1萬元，到後端執行啥的

情況一: ref 定義的資料
	監視 ref 定義的基本類型 資料: 直接寫資料名即可, 監視的是其 value 值的改變

		<template>
		  <div class="person">
		    <h1>監視ref 定義的基本類型 資料</h1>
			<h2>當前求和為:{{sum}}</h2>
			<button @click="changeSum">點我sum+1</button>
			
			
		  </div>
		</template>

		<script lang="ts" setup>
			import { ref,watch } from 'vue'
			
			// 資料
			let sum = ref(0)
			
			// 方法
			function changeSum(){
			  sum.value += 1
			}
			
			console.log(sum.value)
			
			// 監視, 監視ref 定義的基本類型 資料
			const stopWatch = watch(sum,(newValue, oldValue)=>{
			  console.log('sum變化了',newValue, oldValue)
			  if(newValue >= 10){
			    stopWatch() // 結束監視
			  }
			})

		</script>

		    background-color: skyblue;

 		  }

		   margin: 0 5px;
		  }

		    font-size: 20px
		  }
		</style>
