（00:00）課程介紹與定位

	* 主講者是尚硅谷的前端講師張天禹，這套課程是全新 Vue3 系列課程，強調「快速上手」與「短平快」講解方式。

	* 課程內容採用 TypeScript + 組合式 API + <script setup> 語法糖 為主。

（01:04）教學範圍與語法策略

	* Vue3 大部分向下兼容 Vue2，但本課程 優先只使用 Vue3 新語法（避免舊語法混淆）。

	* 核心語法涵蓋：reactive、ref、computed、watch、生命週期等。

（02:07）進階內容與實戰主題

	* 除核心語法外，還會講解 HOX、自定義 ref、路由、Pinia（推測是 pn r meet ）、組件通訊、響應式 API 等常用開發技巧。

	* 面試常見主題也會整理講解。

（02:07）TS 教學筆記支援

	* 課程提供一份 TypeScript 核心內容筆記，對於 TS 較弱的學生可以快速掌握 TS 重點，與課程代碼精準對應。

（03:12）直播錄製形式與互動

	* 全部課程是以 直播方式錄製，直播中會即時回答學生問題，對大家常見疑問進行專門的小節講解。

（03:12）課程會持續更新

	* 若 Vue3 版本有新特性或學生希望講解的技術點，老師會 持續更新與跟進 相應內容。

（04:16）Vue2 vs Vue3 學習建議

	* 推薦學習策略：如果時間充裕可先學 Vue2 再學 Vue3；若急著開發或時間有限，就 優先學習當前項目所用版本。

	* 市場生態逐漸向 Vue3 傾斜，庫與插件多數已完成 Vue3 遷移。
