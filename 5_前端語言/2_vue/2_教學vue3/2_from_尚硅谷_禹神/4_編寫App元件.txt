編寫App元件

	(00:01) 介紹專案結構：指出 index.html 是真正的入口檔案，並提到 main.ts 才是從 src 開始的主戰場。

	(01:11) 解釋 createApp 的意義：把建立前端應用比喻成「造花盆」， createApp() 是用來「創建應用根基」。

	(02:12) 說明 App.vue 是根組件，其他所有組件（A.vue、B.vue 等）都是放在這個根上的子組件。

	(03:15) 介紹 mount('#app') 的作用：把 Vue 根組件掛載到 index.html 中 ID 為 app 的 DOM 容器。

	(05:19) 檢視 src 目錄結構：強調 main.ts 和 App.vue 是 src 中最重要的兩個檔案，其他組件都在 components 資料夾內。

	(06:13) 演示如何刪掉並重建 src：建立新的 main.ts 和 App.vue 來實作學習。

	(08:01) 在 main.ts 裡引入 createApp 和 App.vue，然後呼叫 createApp(App).mount('#app') 來啟動應用。

	(09:03) 解釋 .vue 檔案的三個區塊：

		template（結構）

		script（邏輯/互動）

		style（樣式）

	(10:05) 在 App.vue 裡寫入基本 HTML 結構與 CSS 樣式，並展示 Vue 3 推薦使用 TypeScript 的提示。

	(11:06) 演示如何定義 Vue 組件名稱與結構，接著再次啟動開發伺服器來檢視結果。

	(12:00) 總結三句核心概念：

		index.html 是入口並需包含 Vue 掛載位置；

		入口解析後指向 main.ts；

		Vue3 用 createApp 建立應用實例。
