		$event 到底是啥? 啥時能 .target
		對於原生事件, $event 就是事件對象 => 能 .target
		對於自訂事件, $event 舊式觸發事件時, 所傳遞的資料 => 不能.target

  import App from './App.vue'

  import router from './router'

  // 創建應用

  // 創建 pinia
  const pinia = createPinia()

  // 安裝 插件 pinia

  // 安裝 插件 路由器

  // 掛載應用
  app.mount('#App')
