Arrays.stream 走訪陣列
	// 創建一個整數陣列
	int[] numbers = {1, 2, 3, 4, 5};

	// 使用Stream走訪陣列並印出每個元素
	Arrays.stream(numbers)
		  .forEach(System.out::println);
