概要
	1.手動深層複製： 自己寫邏輯，對集合或自訂對象型別進行逐層複製。

	2.使用序列化（Serialization）： 將對象序列化為二進制流後再反序列化，生成完全獨立的對象（需要對對象實現 Serializable 接口）。

	3.使用第三方庫： 如 Apache Commons Lang3，它的 SerializationUtils 可以輕鬆實現深層複製。

特性
	基本型別和不可變類型（如 int、double、String）會直接複製其值。
	對象型別（如 List、Map 或自訂類別的屬性）只會複製參考（reference），而不是複製實際的對象。
	如果複製的 Bean 中包含可變對象（如集合或其他引用類型），修改原對象中的這些屬性，目標對象也會受到影響，因為兩者共享同一個內存地址。
