第一次開啟瀏覽器
	1.未登入
		嘗試瀏覽未驗證網頁 
			=> 常見預設轉登入頁面或未驗證錯誤頁面
	2.已登入
		嘗試瀏覽登入頁面
			=> 常見預設轉登入後的首頁或強制登出並要求登入
		嘗試瀏覽登入頁面送出到後端的 action by GET (/performLogin get)
			=> 常見轉為 404/未提供指定服務 而不是自動登出

		嘗試瀏覽未授權頁面
			=> 常見預設跳出未授權錯誤
		session 過期(預設三十分鐘)
			=> 常見轉登入畫面並顯示過期資訊
				/login?expire
重啟瀏覽器
	1.之前已登入
		嘗試瀏覽登入頁面
			=> 常見
			
雙開瀏覽器情境(sessionManager 模式)
	1.已於 A 瀏覽器登入使用者a，B 瀏覽器嘗試再登入使用者a可否?
	2.已於 A 瀏覽器登入使用者a，若 B 瀏覽器可登入使用者a，則 A 瀏覽器重新整理會登出?

錯誤測試
	登入頁面
		輸入錯誤帳號或密碼
		輸入錯誤圖形驗證碼
