==========================================================================================
=================================== 下載 from git server =================================
==========================================================================================

第一次從遠端 git server url 下載
	git clone ${url}

從遠端 git repo 下載最新的資料到本地 repo，並且將本地repo資料取出到工作目錄
	git pull

從遠端 git repo 下載最新的資料到本地 repo(本地repo還沒有與工作目錄同步)
	git fetch
