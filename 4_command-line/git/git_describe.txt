查看當前 branch 最近 tag
	git describe --tags --abbrev=0
		// --abbrev=0 代表不要對 commit 的 SHA-1 校驗碼進行縮寫，而是顯示完整的校驗碼。
		// 這裡設置為 0，表示不進行縮寫
