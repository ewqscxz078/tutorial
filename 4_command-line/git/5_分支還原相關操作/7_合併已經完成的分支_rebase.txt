===============================================================================
============================ git rebase =======================================
===============================================================================

將工作目錄(HEAD)的所有紀錄嫁接到 <branchName> 分支上(通常是為了將分支簡化成同一條線上，不想看到分支圖線太複雜用，一般較少使用)
	git rebase <branchName>

中斷 rebase 當衝突時
	git rebase --abort
