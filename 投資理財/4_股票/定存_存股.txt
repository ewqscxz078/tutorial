定存 / 老母雞
	存錢
	固定的銀行利率1%
存股 / 放山雞 => 要找穩定型的老母雞 / 不用擔心股價波動
	存股票 => 股價會波動
	存成股票獲得高於銀行定存利率1%的利息
		=> 股息 => 股利

	台股整體約4%殖利率

	股票利率高於銀行定存利率的原因是需要承擔本金損失的風險


	除權息規定，一年一度
		上市公司發放股利都需要除息，以維持總體市值不變
			除息那天 => 股價會自動減去股利 => 過股價沒有漲超過減去的股利 => 等於倒貼 => 拿自己的錢發股利給自己

		ex:
			中華電信
				7/20 股價100
					假設
						發放現金股利$5元
					若是發放股票則叫 除權，換算對價等值股票
				7/21 除息日 100-5 = 股價自動變 95

				一般正常營運獲利的公司
					會因獲利將減少的股價填回原本除掉的息 => 填息

				過一陣子後
				漲回原本100元
