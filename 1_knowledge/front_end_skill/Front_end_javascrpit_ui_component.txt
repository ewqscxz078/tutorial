前端 javascript ui component

ref openAI
1.React：
	由 Facebook 開發，是一個用於構建用戶界面的 JavaScript 庫。它採用虛擬 DOM 技術，提供了高效的更新機制，
	並支援組件化開發，使得構建大型、可重複使用的 UI 組件更加容易。

2.Angular：
	由 Google 開發，是一個完整的前端框架，提供了許多功能和工具，包括數據綁定、依賴注入、
	路由和模組化等。它強調雙向數據綁定，使得數據和視圖能夠保持同步。

Vue.js：
	一個輕量且易於學習的前端框架，提供了虛擬 DOM 和數據綁定，類似於 React 和 Angular。
	Vue.js 的生態系統不斷擴大，有許多插件和庫可以用於擴展功能。

jQuery：
	儘管不是嚴格意義上的框架，但 jQuery 是一個非常流行的 JavaScript 库，用於簡化 DOM 操作、處理事件和執行 AJAX 請求等。
	它主要用於處理瀏覽器兼容性問題和操作 DOM 元素。

Ember.js：
	一個全功能的前端框架，提供了大量的工具和功能，用於構建高度交互式的 Web 應用程式。
	它強調約定優於配置，使得開發更加快速和一致。

Backbone.js：
	一個輕量級的框架，提供了基本的結構和模型，
	幫助你構建具有結構的 Web 應用程式。

Svelte：
	一個新興的框架，將組件編譯為高效的原生 JavaScript 代碼，而不是在運行時處理虛擬 DOM。
	這使得 Svelte 應用程式的大小更小且性能更好。
