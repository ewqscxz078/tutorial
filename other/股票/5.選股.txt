賺什麼
	依據公司的負債比率參考對應指標
	評估實際獲利能力
		1.EPS (Earnings Per Share) / 每股稅後盈餘 / 每股純益
			 = 稅後淨利(繳完稅之後的淨利) / 股數(資本額/10) = 平均每一股完稅之後還賺多少錢

			通常以季為單位
			四季加起來可得到全年的eps
			年報通常為
				隔年三月才會公布
					(大多數的董事會會在這個時候開會決定拿這些eps 多少比例出來發股利)
					盈餘分配率 : 可能全發或、八成、五成、不發
						依據投資者的類型不同適合不同的盈餘分配率


			看每一股賺多少錢，但不完全代表公司是否賺

			EPS 可參考 台灣股市資訊網查詢 : 股利政策

		2.	ROA (Return On Asset) / 資產報酬率(整體資產的獲利能力) /
			評估該公司整體資產運用效能

			= 稅後淨利 (繳完稅之後的淨利) / 資產(經濟資源:投資於賺錢的資源)
			ex:
				派蝦苑用
					1.500萬 店面
					2.100萬 設備
					3.100萬 蝦苗
					4.300萬 現金
					公司資產 =  1000萬
					若去年一整年淨利200萬元
					roa = 200萬元 / 1000萬 = 20 %資產報酬率

			ROA 可參考 台灣股市資訊網查詢 : 經營績效

		3.ROE (Return On Equity) / 股東權益報酬率
			= 稅後淨利 (繳完稅之後的淨利) / 淨值(股東權益)
			來評估公司透過股東資金獲利的能力

			ex:
				派蝦苑公司資產 = 1000萬
					由 負債500萬(銀行借貸) + 淨值500萬

				roe = 200 萬 / 500萬 = 40%

			公司用跟銀行借來的錢賺錢 =>	融資槓桿

			ROE 可參考 台灣股市資訊網查詢 : 經營績效

算什麼
	目前股價的價格合不合理
		進場時機
	價格脫離不了遛狗理論
		主人牽繩遛狗
			狗會跑來跑去
			但脫離不了主人，最終會回到主人身邊
	價格評估工具
		1.本益比 PER (Price-Earings Ratio)	(賺多少)
			股價與 eps 關係
			股價成本和每股存益之間的比例

			= 股價(購入成本) / EPS(年度每股存益) = 本益比 PER

			ex:			a		b
				股價 	100		100
				EPS		4		5
				本益比	100/4	100/5

				1.跟自己比
				2.跟同產業比

		2.股價淨值比 PBR (Price-Book Ratio) (值多少)
			每股價格(購入成本) / 每股淨值(淨值/股數) = 股價淨值比

			ex:
				派可公司資產1000萬
					負債500萬、淨值500萬
					十萬股(資本額100萬)
					每股淨值 = 500萬 /10萬股 = 50元
					若現在股價40元
						40 / 50 = 0.8

		3.殖利率 yield		(配多少)
