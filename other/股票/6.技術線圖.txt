價格
	價格位在 XY軸，
		K線 	代表Y軸，價格上下相關比較
		均線	代表X軸，價格左右相關比較(與日期)
	k線 candlestick 從燭台延伸而來 / 日k / 日線
		根據交易日成交 開價收價 最高價 最低價 價位顯示

		粗線
			上收 	下開
		細線
			上高	下低

		收盤高於開盤
			k線紅色
				=>紅棒
		收盤低於開盤
			k線綠色
				=>俗稱黑棒
		收開高低階一樣
			k線黃色或白色

		開等於低、收等於高
			=>	粗紅棒
		開等於高、收等於低
			=>	粗黑棒
		收等於開
				=>十字線

		開高收高
				=>下引線

		開低收略上一點低
				=>上引線

		開高收低同一點
				=>一橫線 / 開盤漲停鎖死

	把 日k 一周為一個單位 => 周K

	周K => 月k => 季K => 年K
	大部分用到 季K 之前足夠

	均線
		1.長短天期差異
		2.平均基礎的差異
		最常用的方式有
			5日 (最近一周平均價格)	5MA
			10日(最近兩周平均價格)	10MA
			20日(最近1月平均價格)	20MA
			60日(最近1季平均價格)	60MA
			120日(最近半年平均價格)	120MA
			240日(最近1年平均價格)	240MA
		把每天的平均價格 紀錄並連成一線 => 移動平均線 moving average

		季線不能破，因為往往要一季才能漲回來
數量
	成交量、法人買賣、融資卷
指標
	KD、MACD
	用價格換算指標
		短期指標
			KD	/	隨機震盪指標	/	Stochastic Oscillator
				常被參考進出時機 / 近期價格趨勢高低
				於5MA、10MA不容易拿來參考，因為均價為收盤價的平均換算
				超過九天不適用瞭解價格強落

				原始價格(每日收盤價) => 計算公式	=>	求得數值 kd => 每日移動變化
					來判斷近期股價強弱趨勢

				計算公式
					1.收盤價 => rsv (raw stochastic value)未成熟隨機值	(0~100 ，50為多空標準，)
						(今日收盤價 - 過去九天最低價) / (九天內最高-最低) * 100
					2.rsv 加權
						今日 k
							今日三分之一 rsv
							昨日以前三分之二 k
					3.k值加權再平滑
						今日 d
							今日三分之一 k
							昨日三分之二 d
				常見
					超過 80 代表超買區 / 高檔區
					80-50 代表多
					50-20 代表空
					低於 20 代表超賣區 / 低檔區

		RSI (relative strength lndicator) / 相對強弱指標
			評估買賣雙方力道的消長
			計算公式
				上漲平均(常見5-6天) / (上漲平均+下跌平均) = RSI %

				RSI
					值小 跌均大	=> 賣方強
					值大 漲均大 => 買方強
