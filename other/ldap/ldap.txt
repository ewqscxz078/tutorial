objectclass
	LDAP對象類
Entry
	entry可以被稱爲條目，一個entry就是一條記錄，是LDAP中一個基本的存儲單元
DN(Distinguished Name)
	LDAP中entry的唯一辨別名
LDAP Search filter
	使用filter對LDAP進行搜索。
dc(Domain Component)
	域名
Base DN
	一條Base DN可以是“dc=163,dc=com”，也可以是“dc=People,dc=XX,dc=XX”。
uid
	User ID 用戶id
ou(Organization Unit)
	組織單位，類似於Linux文件系統中的字目錄
cn(Common Name)
	類似於公共名稱
sn
	真實名稱
rdn
	相對辨別名，類似於文件系統中的相對路徑
