螢幕分割快速鍵

	1.如何快速將兩視窗左右分割佔比

		✅ 最快、最實用（強烈建議）

			⌨️ Win + ← / →（左右方向鍵）

				這是最直覺、最穩定的方法。

					操作步驟：

						1.點一下或用 Alt + Tab 切到「應用程式 1」

						2.按
							Win + ← → 視窗固定在 左 50%

						3.Windows 會自動跳出右側可選視窗

						4.選「應用程式 2」
							→ 它會自動佔 右 50%

					📌 完全不需要滑鼠拖拉
					📌 單螢幕、雙螢幕都適用


		🧠 進階但更彈性（Windows 11 新功能）

			⌨️ Win + Z（Snap Layouts）

				Windows 11 專屬，適合之後你想要不是 50 / 50 的情境。

				操作方式：

					1.切到某個視窗

					2.按 Win + Z

					3.螢幕右上會出現版面配置

					4.選擇 左右 50% / 50%

					5.再選另一個視窗填滿剩下區塊

				📌 對於未來想 30 / 70、三分割很有用
				📌 不必移動滑鼠到最大化按鈕

			🔧 系統設定（建議確認一次）

				確認 Snap 功能有開啟（預設是開的）：

					1.設定

					2.系統

					3.多工處理

					4.確保：

						✅ 視窗貼齊（Snap windows）已開

						✅ 當我貼齊視窗時顯示可用視窗


		⚡ 超快切換流程（鍵盤流）

			如果你習慣全鍵盤操作，流程可以是：
				Alt + Tab        （切到 App1）
				Win + ←          （左半）
				Alt + Tab        （切到 App2）
				Win + →          （右半）

			這樣 完全不用滑鼠。


		🧩 進階玩家選項（可選）

			如果你之後有這些需求：

				* 固定某些 App 永遠左 / 右

				* 一鍵還原「工作版面」

				* 開發用（IDE + Browser + Log）

			可以考慮：

				* PowerToys → FancyZones

			但以你現在「單一螢幕、50/50、不想拖拉」
				👉 Win + ← / → 就是最佳解。

	2.快速變成 100%（最大化）

		✅ 最快、最直覺（首選）
			⌨️ Win + ↑（方向鍵上）

			怎麼用：

				1.先點到（或 Alt + Tab 切到）你要放大的那個視窗

				2.按 Win + ↑

				3.視窗立刻變成 100% 全螢幕

			📌 從左右 50/50 → 100%
			📌 不會影響另一個視窗（它會自動退到背景）

		🔁 快速「最大化 ↔ 還原」切換
			⌨️ Win + ↑ / Win + ↓

				* Win + ↑：最大化（100%）

				* Win + ↓：還原成原本大小
				（如果原本是左右貼齊，會回到貼齊狀態）

			這在「左右分割 ↔ 全螢幕」來回切非常順。

		🧠 不碰方向鍵的方式
			⌨️ Alt + Space → X

				1.切到該視窗

				2.按 Alt + Space

				3.再按 X（最大化）

			📌 這是老 Windows 通用招
			📌 適合鍵盤在主鍵區、不想移手的情況

	⚡ 超快鍵盤流程（實戰）

		假設你現在是：

			左邊：IDE
			右邊：Browser

		想快速放大 IDE：
			Alt + Tab   （切到 IDE）
			Win + ↑     （全螢幕）

		回到左右：
			Win + ↓

	總結
		| 目的        | 快捷鍵        |
		| ----------- | ------------- |
		| 左 / 右 50% | `Win + ← / →` |
		| 100% 全螢幕 | `Win + ↑`     |
		| 還原        | `Win + ↓`     |
