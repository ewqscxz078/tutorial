被法律承認的機構團體分類，佔股市總交易量的比重很高，會大幅影響股市的漲跌
透過三大法人動向來預測股價，也屬於籌碼面分析的其中一種分析方式

法律定義
	自然人:一般人(你、我、他)，享有基本的人權
	法人:公司、營利組織機構，法律上賦予權力的團體。

三大法人
	外資：指台灣以外的國外投資機構。
	投信：指國內的證券投資機構，募集投資人的錢來操作。
	自營商：證券公司自己的投資部門，可以和證交所直接買賣

外資	(國外來的錢)	=> 證卷商	=> 間接買賣		=>	證卷交易所
投信	(投資人的錢)	=> 證卷商	=> 間接買賣		=>	證卷交易所
自營商	(公司的錢)		=> 直接買賣					=>	證卷交易所


三大法人特性(資料整理：Mr.Market市場先生)
		定義						策略											資金
外資	指台灣以外的國外投資機構	重視基本面、長期投資，以大型股為標				資金佔比、成交量都是股市中最高

投信	指國內的證券投資公司		偏好波段操作，短中期投資，以中小型股為標的		資金佔比、成交量較外資少

自營商	證券公司自己的投資部門，	偏好短期策略、股價起伏較大、能快速獲利標的為主	資金佔比、成交量比外資、投信少
		用公司自有資金來投資

三大法人買超，上漲機率大；三大法人賣超，下跌機率大。
	三大法人買超：三大法人買進的數量、金額，超過賣出的數量或金額，意思就是法人看好這檔標的，也就是散戶賣得多。
	三大法人賣超：三大法人賣出的數量、金額，超過買進的數量或金額，意思就是法人看壞這檔標的，也就是散戶買得多。
