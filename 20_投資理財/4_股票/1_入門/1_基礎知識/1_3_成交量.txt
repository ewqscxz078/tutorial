成交量(Trading volume)與股票關係
	新手看價、老手看量、高手看籌碼

	價量關係、跟持股張數有關係(若該檔交易量多餘持股張數則不影響，反之若小於則會有影響)

	k線 => 方向盤(趨勢方向)

	成交量 => 油門(股市是否熱絡)、流動性

買賣
	賣方價位固定，買方只能決定買或不買

成交
	買賣雙方出價，謀合的結果

一家股票數量是固定的


成交數越高
	雙方交易數大，出價落差大容易謀合，越不一致
	反之出價差不多，交易數少


成交量要看雙方積極程度 => 股價有相對反映狀況
	兩邊冷、一邊冷一邊熱無法搓合成交量
	只有兩邊熱才能搓合成交量

量能名詞
	短期回檔趨勢中的「支撐量」和「防守量」、推升股價向上的「攻擊量」、短期窒息的「凹洞量」、高檔的「出貨量」
	中期趨勢中的「底部支撐量」、行進間「換手量」、中期整理「窒息量」、上攻量能不足時的「補量」、觀察進攻發起點「多空量」型態
	「突兀量」的發生如在底部區，可觀察兩天，股價若遠離底部區可視為「底部支撐量」。
	若股價為「盤整型態」，需觀察一週以上變化；「突兀量」的低點若跌破，將造成底部支撐失敗之「萬里長空」走勢。
	所以盤跌過久的股票一但出現異常大量，不一定是「支撐量」。

	量的揮灑過程中，早期有「凹洞量」、中期就有「換手量」、晚期也有「出貨量」

成交量顯示的顏色代表什麼？
	成交量柱狀圖顯示的顏色是用來區分「昨日收盤價」與「今日收盤價」之間的關係。

	今日收盤價＞昨日收盤價：成交量顯示紅色。
	今日收盤價＝昨日收盤價：成交量顯示黑色或黃色（不同看盤軟體顯示的顏色可能不同）。
	今日收盤價＜昨日收盤價：成交量顯示綠色。

成交量十大規則:反映主力意圖
	1.縮量上漲 => 還會上漲
	2.縮量下跌 => 還會下跌
	3.低位放量 => 後勢看漲
	4.高位放量不漲 => 頭部出現
	5.低位放巨量不漲 => 還會回調
	6.低位縮量不跌 => 底部出現
	7.高位放巨量上漲 => 還會下跌
	8.底位放量下跌 => 後市很難再創新高
	9.頂部無量下跌 => 後市還會新高
	10.底部價量齊升 => 說明趨勢已經形成
