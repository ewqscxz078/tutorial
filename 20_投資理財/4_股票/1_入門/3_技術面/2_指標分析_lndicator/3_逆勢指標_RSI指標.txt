RSI
	1.判斷股價相對強弱
		上漲視為買方力道，下跌視為賣方力道
	2.以某段時間 收盤價的 平均漲幅 與 平均跌幅 所計算出來的數值，可看出股價相對強弱的指標

用價格漲跌幅換算指標
	短期指標
		RSI (relative strength lndicator) / 相對強弱指標
			評估買賣雙方力道的消長
			計算公式
				上漲平均(常見5-6天) / (上漲平均+下跌平均) = RSI %

				N日平均漲幅 / (N日平均漲幅-N日平均跌幅) * 100
				N日平均漲幅 = N日內上漲總幅度/N天數
				N日平均跌幅 = N日內下跌總幅度/N天數

				RSI
					值小 跌均大	=> 賣方強
					值大 漲均大 => 買方強

公式代表了三件事
	1.RSI的範圍一定在 0-100之間
	2.RSI越大，代表過去「漲的機率」越大
	3.如果漲和跌的機率一樣，RSI會是中間值 50

常見問題
	1.RSI值越大越好嗎？
		RSI又被稱為「逆勢指標」
			代表很多時候你要反著來！也就是 RSI太大的時候，就要小心是不是市場過熱

	2.RSI 有6日、14日？差在哪？
		常用的天數有 6日、9日、12日、14日等，
		通常短週期的 RSI值變動比較大 (或說靈敏)，
		長週期的 RSI值變動比較小 (或說遲鈍)。
		市場上也有人會將
		短週期和長週期的 RSI值 做成連續線圖，
		用來判斷黃金交叉、死亡交叉

	3.RSI 跟 KD 差在哪？
		KD的計算方式，
			是拿今天的股價 和過去幾天相比是強勢還弱勢
			所以是用來判斷 當天股價在相對高點還是低點

		RSI則
			是拿 最近幾天的股價漲、跌幅
			來計算股價買、賣方力道的比例
			所以是用來判斷 這段期間的買、賣方力道強弱

RSI 用法
	1：判斷買賣時機
		投資人會透過觀察兩條 RSI線
		來判斷黃金交叉、死亡交叉
		分別是 短週期RSI 和長週期 RSI
		看盤軟體上 會標註短週期和長週期的天數
		有些軟體短週期是 6天、長週期是 12天
		有些軟體則是短週期是 5天、長週期是 10天

		黃金交叉：買進訊號
			當「短週期」向上突破「長週期」
			一般稱為「黃金交叉」，
			代表短期內上漲力道夠，是買進的訊號，
			建議輔以其他指標來判斷是否要進場。
		死亡交叉：賣出訊號
			當「短週期」向下跌破「長週期」
			一般稱為「死亡交叉」，
			代表短期內下跌力道強，是賣出的訊號，
			一樣建議輔以其他指標來判斷是否要出場。

	2：判斷市場熱度
		RSI > 80時，為「超買」訊號，市場過熱，要準備開始跌了。 : 80以上考慮站在賣方
		RSI < 20時，為「超賣」訊號，市場過冷，要準備開始漲了。 : 20以上考慮站在買方
		雖然這裡是以 80、20 為超買、超賣訊號
		但也有人以 70、30 為超買、超賣訊號
		所以要特別說明，這部分的標準並不絕對
		建議大家還是要參考其他的指標 綜合評估

RSI 使用限制
	1.遇到盤整期時，RSI會不斷出現黃金、死亡交叉，效果會打折

	2.如果遇到大漲、大跌，RSI值的波動會較劇烈

	3.容易在股價極度強勢與弱勢時出現鈍化，影響進、出場時機的判斷

	4.從公式中可以看出來，RSI看的是 N 天的狀況，因此只適用於判斷短期多空力道。


鈍化 :
	RSI 連續三天站上 80 => 高檔鈍化、搭配關鍵五日均線後符合則容易變飆股，站上五日均線下代表鈍化結束反轉向下
	RSI 連續三天站上 20 => 低檔鈍化、搭配關鍵五日均線前符合則容易變垃圾股，站上五日均線上代表鈍化結束反轉向上


背離
	牛市背離: 指數創低，RSI未創低
	熊市背離: 指數創高，RSI未創高
