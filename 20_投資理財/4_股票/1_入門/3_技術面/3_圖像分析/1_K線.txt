價格
	價格位在 XY軸，
		K線 	代表Y軸，價格上下相關比較
	k線 candlestick 從燭台延伸而來 / 日k / 日線
		根據交易日成交 開價收價 最高價 最低價 價位顯示 : 了解這支股票 週期(日、周、月、季、年)K 的波動狀況

		粗線
			上收 	下開
		細線
			上高	下低

		收盤高於開盤
			k線紅色
				=>紅棒
		收盤低於開盤
			k線綠色
				=>俗稱黑棒
		收開高低階一樣
			k線黃色或白色

		開等於低、收等於高
			=>	粗紅棒
		開等於高、收等於低
			=>	粗黑棒
		收等於開
				=>十字線

		開高收高
				=>下引線

		開低收略上一點低
				=>上引線

		開高收低同一點
				=>一橫線 / 開盤漲停鎖死

	把 日k 一周為一個單位 => 周K

	周K => 月k => 季K => 年K
	大部分用到 季K 之前足夠


單看k線圖形不太夠，還要再考量 量、價、位階
