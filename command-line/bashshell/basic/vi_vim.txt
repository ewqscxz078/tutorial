===================================================================
========================= vi/vim ==================================
===================================================================
vi -b 顯示換行符號

gg 到第一頁
shift g 最後一頁
ctrl + f 下一頁
ctrl + b 上一頁

不儲存離開 vi： :q
不儲存強制離開 vi： :q!
儲存且離開 vi ： :wq
將檔案還原到最初狀態(進入 vi 後一開始的樣貌) ： :e!

刪除
	刪除一行
		dd
	刪除範圍行數
		:[start line],[end line]d

dd 並游標到某行最後面後按下 p ，則會插入下一行

copy
	yy 複製游標該行
	游標到某行最前面後按下 p ，則會插入下一行

search
	next n
	before N

跳到指定行數
	:n，代表跳?到第n行

還原上一步
	u

查詢 tab 符號
	/\t

取代
	:%s/{關鍵字}/{取代後的值}/g
	ex
		:%s/\t/    /g

===================================================================
========================= 額外設定 ================================
===================================================================
顯示行數
	:set number
隱藏行數
	:set nonumber

不分大小寫搜尋
	:set ignorecase
取消不分大小寫搜尋
	:set noignorecase

檢視不可見特殊字元
	:help digraph-table

顯示隱藏字元
	:set list

不顯示隱藏字元
	:set nolist

設定顯示哪些隱藏字元
	:set listchars=eol:$,tab:>-,trail:~,extends:>,precedes:<
