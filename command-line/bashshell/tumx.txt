開啟 多功終端機	
	tmux
	
視窗開啟自訂 session-name	
	tmux new -s session-name # ex: tmux new -s irc	

重新命名 session-name
	tmux rename-session [-t current-name] [new-name]
	
建立新的 terminal 在同一個畫面
	Ctrl + a + c

視窗命名
	Ctrl + a ,

切換同畫面下一個的 terminal
    Ctrl + a + n
	
切換同畫面上一個的 terminal
	Ctrl + a + p

同畫面橫切
	Ctrl + a s
	
同畫面直切
	Ctrl + a v (或 C-a %)
	
調整畫面
	Ctrl + a alt + 方向鍵	
	
關閉當前分割的畫面
	Ctrl + d
	
脫離
	Ctrl + a + d
	
回來 tmux
	tmux ls
	tmux attach -t 0 # 參考 tmux ls 取得
	
刪除 session	
	tmux kill-session -t <session_name>	
	
刪除 全部 session		
	tmux kill-session -a
