/usr/sbin/apachectl其實是個腳本；
/usr/sbin/httpd才是真正的程序；

Normal restart(中斷現用的已經開啟的連線，影響線上正在跑的會被kill調，也就是中斷)
	apachectl -k stop
	apachectl -k start
	apachectl -k restart

Graceful Restart(不會中斷現用的已經開啟的連線，若跑完則reRead config，不影響線上正在跑的)
	apachectl -k graceful
	httpd -k graceful

確認版本
	apache2ctl -V // detail
	httpd -v

存取限制
	限制所有存取
		Require all denied
	允許所有存取
		Require all granted
	允許所有在 deyu.wang 網域的主機存取
		Require host deyu.wang

	以 ip 限制存取
		完整 ip
			Require ip 192.168.1.140 192.168.1.141
		部分 ip，允許指定網段
			Require ip 10 172.20 192.168.2
		網段/遮罩，允許指定網段
			Require ip 10.1.0.0/255.255.0.0
		網段/遮罩數字，允許指定網段。
			Require ip 10.1.0.0/16
	以 host 限制存取
		Require host 192.168.*

確認apache服務
	For Ubuntu:
		service apache2 status
