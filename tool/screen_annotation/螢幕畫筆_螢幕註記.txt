螢幕畫筆 / 螢幕註記（screen annotation）」工具
	（教學、畫重點、截圖、浮動標註）。


🧑‍🏫 一、最推薦（教學用完整型）
	🟢 IPEVO Annotator（免費、很強）
		ref https://tw.ipevo.com/pages/annotator-download

	特點（幾乎完全符合你需求）：

		* 可以直接在螢幕畫畫、寫字、框選
		* 有橡皮擦 / 一鍵清除
		* 支援截圖、錄影
		* 有：
			* 放大鏡
			* 聚光燈
			* 白板模式
		* 甚至有「畫完自動消失」功能（不用手動清）

		👉 適合：

			* 線上教學
			* Demo 系統操作
			* 錄教學影片

	👉 重點：
	👉 幾乎就是「數位白板 + 螢幕標註 + 教學工具」整合版

🧑‍🏫 二、工程師 / 簡報常用（超輕量）
	🔵 ZoomIt（微軟出品）

	👉 特點：

		* 快捷鍵啟動（Ctrl+1 放大 / Ctrl+2 畫筆）
		* 可在畫面上畫線、寫字
		* 支援：
			* 螢幕放大
			* 計時器
			* 黑板模式
		* 幾乎無延遲、超輕量

	👉 適合：

		* 技術簡報
		* Demo 系統（像你這種架構說明很適合）
		* 不想裝重軟體的人


🧑‍🏫 三、功能型畫筆（可客製化）
	🟣 ppInk / gInk

	👉 特點：

	* 支援：
		* 畫筆、圖形、文字
		* 編號標記（①②③）
		* 套索選取（可以移動註記）
	* 可高度自訂筆刷

	👉 適合：

		* 想做比較「精緻教學標註」
		* UX / 設計說明

🧑‍🏫 四、極簡工具（免安裝）
	🟡 Pointofix

	👉 特點：

		* 開啟就能畫（超簡單）
		* 有基本：
			* 畫線
			* 框選
			* 打字
		* 很適合臨時教學

	👉 缺點：

		* 功能較少
		* 畫面凍結後要清除才能操作


🏆 給你的建議（工程師 + 架構教學）

	依你平常做：

		* 架構說明
		* OAuth / Gateway flow
		* Demo 系統

	👉 我會這樣搭配：

	🥇 最推薦

	👉 IPEVO Annotator + ZoomIt

		* Annotator → 畫圖、講流程
		* ZoomIt → 放大細節（debug / code）


💡 額外補充（很多人不知道）

	其實：

		* Zoom / Teams 本身有畫筆，但很弱
		* Google Meet 幾乎沒有（才需要這些工具）

	👉 所以「外掛畫筆工具」才是主流
