不建議再使用
	Java 1.0
		package path : java.util
			Date
				具有許多方法，但存在一些問題，例如不是線程安全的，並且其年份是從1900年開始，月份是從0開始

	Java 1.1
		package path : java.util
			java.util.Calendar
				用於處理日期和時間
				解決了一些 Date 的問題，提供了更多功能，但仍然存在一些複雜和容易出錯的地方
				Calendar 是可變的，而且也不是線程安全的

Java 8
	package path : java.time
		主要目標是提供更清晰、更易用且更安全的日期和時間處理方式。
			LocalDateTime
				表示日期和時間，不包含時區。
					適用於需要處理完整日期和時間的情境。
			LocalDate
				表示日期，不包含時間。
					不可變的，並且提供了簡單易用的方法來處理日期。適用於不需要時間信息的日期操作
			LocalTime
				表示時間，不包含日期。
					適用於需要處理時間而不關心日期的情境

			YearMonth
				表示年和月，不包含日期和時間。
					適用於處理僅包含年和月的情境，沒有日期或時間的概念

			ZonedDateTime
				表示帶有時區的日期和時間。
					適用於需要處理時區信息的情境。
			Duration
				表示時間上的持續時間，以秒和納秒為單位。
				適用於計算兩個時間點之間的差異。
			Period
				表示日期上的持續時間，以年、月、日為單位。
				適用於計算兩個日期點之間的差異。
